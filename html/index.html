Ce projet est le TP d'INF224 de Pascal Rouppert. Il propose une interface graphique programmée en java. Cette interface peut être lancée via la commande make run-interface. Elle communique avec un server, qui doit être lancer en parallèle via la commande make run-server . Ce serveur crée une table d'objets multimédia à partir du dossier Multimedia, dont le contenu est décrit par le fichier Database.txt.

    Commandes utiles :
  • make run-server : lance le serveur
  • make run-client : lance le client
  • make run-interface : lance l'interface graphique
    • make run-interface
  • make clean-all : détruit tous les fichiers objets
  • make all : compile tous les fichiers
    Précision sur le formatage de Database.txt :
  • Une ligne par fichier.
  • pour les vidéos : Video "Durée de la vidéo" "Nom de la vidéo" "Chemin"
  • pour les photos : Photo "Longitude" "Latitude" "Nom de la Photo" "Chemin"
